# 🐾 PetNet — Prototipagem UI/UX

O **PetNet** é uma plataforma mobile-first focada na conexão direta, simples e gratuita entre doadores (pessoas físicas ou ONGs) e adotantes de animais de estimação, funcionando como um "OLX da Adoção".

## 🚀 Protótipo ao Vivo

O protótipo interativo e navegável da interface foi gerado via Lovable e pode ser acessado no link abaixo:

🔗 **[Acessar Protótipo](https://conexao-pet-amada.lovable.app)**

## 📱 Funcionalidades Mapeadas (11 Telas)
- **01.** Tela Inicial (Descoberta e Filtros)
- **02.** Login de Usuários
- **03.** Cadastro em Duas Etapas (PF vs. ONG)
- **04.** Listagem de Animais (Busca Avançada)
- **05.** Detalhes do Anúncio (Carrossel e Informações)
- **06.** Publicação de Novo Animal (Validação de Dados)
- **07.** Painel do Usuário (Dashboard de Métricas)
- **08.** Gerenciamento de Anúncios (Mudança de Status)
- **09.** Chat / Mensagens Integradas
- **10.** Canal de Denúncias (Segurança)

## 🛠️ Tecnologias de Interface
- React.js + TypeScript
- Tailwind CSS (Dark Mode Nativo)
- Vite
